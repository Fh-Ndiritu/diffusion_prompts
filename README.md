# Searching Prompts

## SETUP
1. Clone Repo
2. setup .env(from env.example)
3. docker compose up --build
4. ./run rails db:setup
5. accesible on https://localhost:8000/

# TESTING
./run rspec

1. Rails 7.1.2
2. Ruby 3.2.2
3. Postgresql
4. Docker
5. Elasticsearch
6. Rspec
7. Capybara/Selenium
